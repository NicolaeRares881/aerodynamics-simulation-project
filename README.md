# Aerodynamics Simulation Project

## 🚀 Overview

This project simulates basic vehicle acceleration with aerodynamic drag effects using Python.

It models:

- Vehicle acceleration
- Air resistance (drag)
- Speed limitation due to aerodynamic forces

---

## 🧠 Physics Concepts Used

- Newton's Second Law (F = m × a)
- Aerodynamic drag formula:
  
  F_drag = 0.5 × ρ × v² × Cd × A

- Numerical simulation using time steps

---

## 💻 Technologies Used

- Python
- Matplotlib
- NumPy
